# AI Neuro Dev Notes

Public notes and small examples at the intersection of artificial intelligence,
neuroscience, and developer tooling.

## Purpose

This repository is a working notebook for practical ideas that are useful to
students, builders, and researchers who move between AI systems, cognitive
science, and software engineering.

## Contents

- `notes/`: short explainers and bridge notes.
- `examples/`: small runnable or readable examples, including a
  [reading-note template](examples/reading-note-template.md).
- `resources.md`: curated links with context notes.

## Project Principles

- Keep notes concise and source-aware.
- Prefer reproducible examples over vague claims.
- Separate learning notes from experiments.
- Add links only when they are useful enough to revisit.

## Roadmap

- Add short notes connecting AI concepts with neuroscience concepts.
- Add small examples for data workflows and developer tooling.
- Build a curated resource list for AI, neuroscience, and engineering practice.
- Collect project ideas that can grow into useful public demos.

See [ROADMAP.md](ROADMAP.md) for the current project plan.
