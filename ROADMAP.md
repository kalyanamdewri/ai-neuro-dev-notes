# Roadmap

This roadmap keeps the repository focused on small, useful additions instead of
large unfocused notes.

## Near Term

- Add bridge notes that compare AI concepts with neuroscience concepts.
- Add lightweight examples for data cleaning, notebook hygiene, and model notes.
- Expand `resources.md` with links that include short context summaries.

## Mid Term

- Add a glossary for terms shared across AI and neuroscience.
- Add tiny examples that show how to turn reading notes into reproducible code.
- Collect public datasets that are useful for learning computational
  neuroscience.

## Maintenance

- Keep links current.
- Prefer short notes that can be reviewed quickly.
- Move unfinished ideas into issues before turning them into files.
