# Resources

Curated references for AI, neuroscience, and developer tooling.

## AI

- [OpenAI Cookbook](https://developers.openai.com/cookbook): practical
  examples for building with language models and related AI APIs.
- [Hugging Face Learn](https://huggingface.co/learn): hands-on lessons for
  transformers, model use, and modern ML workflows.
- [Hugging Face Papers](https://huggingface.co/papers/trending): a useful way
  to scan recent papers and connect them with community discussion.

## Neuroscience

- [Neuromatch Computational Neuroscience](https://compneuro.neuromatch.io/):
  open computational neuroscience lessons with code-first tutorials.
- [Allen Brain Map](https://brain-map.org/): public brain atlases, datasets,
  and tools for exploring brain structure and function.
- [The Brain from Top to Bottom](https://thebrain.lecerveau.ca/): accessible
  neuroscience explanations with multiple depth levels.

## Developer Tooling

- [GitHub Docs](https://docs.github.com/en): reference for issues, pull
  requests, Actions, and repository collaboration.
- [Python Packaging User Guide](https://packaging.python.org/en/latest/):
  practical guidance for packaging and publishing Python projects.
- [MDN Web Docs](https://developer.mozilla.org/en-US/): reliable reference for
  web platform concepts, browser APIs, and JavaScript.

## How to Use This List

Each link should earn its place by being useful for learning, implementation, or
research follow-up. Add a short note when adding a resource.
