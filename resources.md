# Resources

Curated references for AI, neuroscience, and developer tooling.

## AI

- OpenAI Cookbook: practical examples for building with language models.
- Hugging Face Course: hands-on transformer and model deployment lessons.
- Papers with Code: useful for connecting papers to reproducible baselines.

## Neuroscience

- Neuromatch Academy: computational neuroscience and deep learning materials.
- Allen Brain Map: public brain atlas data and tools.
- The Brain from Top to Bottom: accessible neuroscience explanations by level.

## Developer Tooling

- GitHub Docs: reference for issues, pull requests, Actions, and project work.
- Python Packaging User Guide: practical guidance for packaging Python projects.
- MDN Web Docs: reliable reference for web platform concepts.

## How to Use This List

Each link should earn its place by being useful for learning, implementation, or
research follow-up. Add a short note when adding a resource.
